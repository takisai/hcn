# このプログラムについて

このプログラムは、高度合成数([Wikipedia](https://ja.wikipedia.org/wiki/%E9%AB%98%E5%BA%A6%E5%90%88%E6%88%90%E6%95%B0))を列挙します。

## 必要なもの

外部ライブラリーは一切使用していません。

## 使い方・使用例

    $ make
    $ hcn
    1 1
    2 2
    3 4
    4 6
    5 12
    ...
